Writing own Queries in Spring Data JPA 

@Query : is used to write own Query
@Param : is used to bind formal variable to query parameter
Two types of query can write
	we can write JPQL(JAVA persistant Query) queries similar to HQL 
	Native Query
	//JPQL Query
	@Query("select u from User u where u.name= :n")
	public List<User> getUserByName(@Param("n") String name);
	
    //native Query
	@Query(value="select * from user",nativeQuery = true)
	public List<User> getUsers();
